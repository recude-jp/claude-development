## 作業はすべて Claude Cowork だけで完結します

GitHub のアカウント・GitHub Desktop・ターミナルは一切不要です。

---

## はじめにやること（最初の一回だけ）

システムアドバイザーから以下の2つを受け取ってください：

- **リポジトリURL**（例: `https://github.com/xxx/yyy`）
- **アクセスキー**（英数字の長い文字列）

### 手順

1. **Claude Cowork を開く**（フォルダの選択は不要）

2. 以下のように話しかける：

   > 「セットアップをお願いします」

3. Claude が「リポジトリURL」と「アクセスキー」を聞いてきます。受け取った内容を貼り付けてください。

4. Claude が自動でセットアップを行い、「完了しました。このフォルダを開いてください: ○○」と教えてくれます。

5. Cowork の**フォルダ選択**で、Claude が指定したフォルダを選択する。

これで準備完了です。

---

## 毎回の作業

1. **Claude Cowork を開く**
2. **上で選択したフォルダを選ぶ**
3. 以下のように話しかける：

   > 「業務を自動化したいです」

あとは Claude の質問に答えるだけです。
ファイルの編集もシステムアドバイザーへの共有も、Claude が自動でやります。

---

## 困ったときは

| 状況 | 対処 |
|---|---|
| Claude の言っている意味がわからない | 「もっとかんたんに説明してください」と伝える |
| うまく進まない・止まってしまった | Cowork の画面をスクリーンショットしてシステムアドバイザーに送る |
| エラーメッセージが出た | そのままシステムアドバイザーに共有する（自分で解決しなくてOK） |

「わからないことを聞く」のはこのプロセスの正式な一部です。早めに確認してください。
