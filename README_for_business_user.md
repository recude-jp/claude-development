## はじめにやること（最初の一回だけ）

システムアドバイザーからメールを受け取ったら、以下の手順で進めてください。

### ⚠️ Norton（ウイルス対策ソフト）をお使いの方へ

Claude が作成するファイルが誤検知で削除される場合があります。最初に除外設定をしてください。

**Norton の場合：**
1. Norton を開く → 設定 → ウイルス対策 → スキャンの除外
2. `書類` → `Claude` → `Projects` フォルダを除外に追加

Norton 以外のウイルス対策ソフトをお使いの場合も、同様に `~/Documents/Claude/Projects/` フォルダを除外してください。

---

### 手順

1. **Claude Cowork を開く**

2. 左メニューの「**プロジェクト**」を選び、右上の「**新規プロジェクト**」ボタンを押す

3. 表示されたダイアログに以下を入力して「**作成**」を押す
   - **名前**: 自由につけてOKです。わかりやすくやる気が湧いてくる名前を！（メールに例が記載されています）
   - **手順**: 空白のままでOK
   - **プロジェクトの場所**: メールに記載されたパスを入力してください

4. 作成したプロジェクトを開き、メールの「**チャットに貼り付けるメッセージ**」をそのままチャットに貼り付けて送信する

   ⚠️ プロジェクトを作成しただけでは何も起きません。必ずチャットにメッセージを送ってください。

Claude が自動でセットアップを行い、作業できる状態になります。

これで準備完了です。

---

## 毎回の作業

1. **Claude Cowork を開く**
2. 左メニューの「**プロジェクト**」から、このプロジェクトを選ぶ
3. 以下のように話しかける：

   > 「業務を自動化したいです」

あとは Claude の質問に答えるだけです。
ファイルの編集もシステムアドバイザーへの共有も、Claude が自動でやります。

## 困ったときは

| 状況 | 対処 |
|---|---|
| Claude の言っている意味がわからない | 「もっとかんたんに説明してください」と伝える |
| うまく進まない・止まってしまった | Cowork の画面をスクリーンショットしてシステムアドバイザーに送る |
| エラーメッセージが出た | そのままシステムアドバイザーに共有する（自分で解決しなくてOK） |

「わからないことを聞く」のはこのプロセスの正式な一部です。早めに確認してください。
