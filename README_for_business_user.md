## はじめにやること（最初の一回だけ）

### 1. GitHub Desktop をインストールする

https://desktop.github.com/ を開いてダウンロード・インストールしてください。

### 2. GitHub にサインインする

GitHub Desktop を起動 → **Sign in to GitHub.com** → ブラウザでサインインして完了。

> GitHub アカウントをまだ持っていない場合は、システムアドバイザーに連絡してください。

### 3. このリポジトリをクローンする

1. GitHub Desktop のメニュー → **File → Clone Repository**
2. **URL タブ** を選択
3. このページの URL（`https://github.com/...`）を貼り付ける
4. Clone ボタンをクリック

### 4. Claude Cowork を開いてフォルダを選択する

1. Claude Cowork アプリを起動
2. 「フォルダを選択」でクローンしたフォルダを選ぶ
   （GitHub Desktop の **Show in Finder** ボタンでフォルダを確認できます）

### 5. 初回セットアップを行う

Cowork の画面で以下のように話しかけてください：

> 「セットアップをお願いします」

Claude が「アクセスキー」の入力を求めます。
**システムアドバイザーから別途受け取ったアクセスキーを貼り付けてください。**

「設定が完了しました」と表示されたらセットアップ完了です。

---

## 毎回の作業

Claude Cowork を開き、フォルダを選択して、こう話しかけるだけです：

> 「業務を自動化したいです」

あとは Claude の質問に答えていけば進みます。
ファイルの編集も GitHub への共有も、Claude が自動でやります。

---

## 困ったときは

| 状況 | 対処 |
|---|---|
| Claude の言っている意味がわからない | 「もっとかんたんに説明してください」と伝える |
| うまく進まない・止まってしまった | Cowork の画面をスクリーンショットしてシステムアドバイザーに送る |
| エラーメッセージが出た | そのままシステムアドバイザーに共有する（自分で解決しなくてOK） |

「わからないことを聞く」のはこのプロセスの正式な一部です。早めに確認してください。
