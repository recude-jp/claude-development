# 業務自動化サポート テンプレート

業務ユーザーが Claude と対話しながら要件定義・基本設計を進め、システムアドバイザーが GitHub 経由でレビューするプロセスを管理するテンプレートリポジトリです。

---

## 目的

- 非エンジニアの業務ユーザーが、Git や GitHub を意識せずに自動化プロジェクトを進められるようにする
- Claude がファシリテーターとして対話しながらドキュメントを作成・更新する
- システムアドバイザーが GitHub 経由で設計レビューを行い、品質を担保する

---

## ファイル構成

```
.
├── README.md                      # 本ファイル（テンプレート説明。業務ユーザーのリポジトリからは削除）
├── CLAUDE.md                      # Claude への操作指示（セッション開始時の挙動・各ステップの進め方）
├── SETUP_for_advisor.md           # システムアドバイザー向け初期セットアップ手順
├── README_for_business_user.md    # 業務ユーザー向けプロセスガイド
├── 01_requirements_brainstorm.md  # ステップ①：要件ブレストテンプレート
└── 02_basic_design.md             # ステップ②：基本設計書テンプレート
```

> ステップ③のレビューチェックリスト（`03_design_review_checklist.md`）と
> ステップ④の実装指示書（`04_coding_instruction.md`）はプロセス中に作成します。

---

## 使い方

このリポジトリはテンプレートとして使用します。
新規プロジェクトの開始手順は `SETUP_for_advisor.md` を参照してください。
